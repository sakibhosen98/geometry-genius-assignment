3. What is event bubble?

Event bubble is an javasrcipt concept. This happens when an element receives an event and that event bubbles (or you can say sent or propagated) to its parent and ancestor elements in the DOM tree until it reaches the parent element.

4. What is Callback function?

A callback function is a function passed into another function as an argument.
